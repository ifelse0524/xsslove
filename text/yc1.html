Vscode多行前后缩进

xiaoshishi

2022-5-20 13:30:40 1024 分类专栏: vscode

vscode缩进快捷键:

所谓批量缩进就是选中一段文本代码后，批量让其向左移动或者向右移动。
首先选中文本，后按住以下组合键
Ctrl + [ 或者 Shift+Tab //向左缩进
Ctrl + ] 或者 Tab //向右缩进
实现文本的向左移动或者向右移动
这跟格式化不太一样，不过格式化倒也可以批量处理缩进
但批量缩进并不会对选中的代码进行格式化，只是会实现选中块代码左移或者右移，各有使用场景。

Visual Studio Code格式化代码可通过以下快捷键:

On Windows Shift + Alt + F
On Mac Shift + Option + F
On Ubuntu Ctrl + Shift + I
